# Hangman Game

## Description
This is a simple Hangman Game developed using Python. The program randomly selects a word and the user tries to guess it letter by letter. The player has limited attempts to guess the correct word.

## Features
- Random word selection
- User letter guessing
- Limited attempts
- Win and lose conditions

## Technologies Used
- Python 3

## How to Run
1. Open terminal in the project folder.
2. Run:
   python hangman.py
3. Enter letters to guess the word.
