# Basic Chatbot

## Description

This is a simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs and can hold a basic conversation.

## Features

- Greets the user
- Answers simple questions
- Responds to predefined commands
- Exit option using "bye"

## Technologies Used

- Python 3

## How to Run

1. Open terminal in the project folder.
2. Run:

```bash
python chatbot.py
```

3. Type messages and interact with the chatbot.
4. Type "bye" to exit.

## Sample Commands

- hello
- how are you
- what is your name
- bye
