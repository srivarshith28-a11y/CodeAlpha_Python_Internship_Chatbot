# Stock Portfolio Tracker

## Description
This is a simple Stock Portfolio Tracker developed using Python. The program allows users to enter stock names and quantities, then calculates the total portfolio value based on predefined stock prices.

## Features
- Predefined stock prices
- User input for stock names and quantities
- Portfolio value calculation
- Simple and easy-to-use interface

## Technologies Used
- Python 3

## How to Run

1. Open terminal in the project folder.
2. Run:

```bash
python stock_tracker.py
```

3. Enter stock names and quantities when prompted.
4. View the total portfolio value.

- TCS
- INFY
- WIPRO
- RELIANCE
